✅ 目标：CSS 变量 + 白底
// 统一使用 CSS 变量
// 这些变量已在 globals.css 定义

background: var(--bg-deep)
// → #faf7f2 (温暖米白)

background: var(--bg-card)
// → #ffffff (纯白卡片)

color: var(--text-primary)
// → #1c1917 (暖黑)

color: var(--text-secondary)
// → #78716c (暖灰)

border: var(--border-subtle)
// → rgba(0,0,0,0.06)

accent: var(--accent)
// → #d97706 (温暖琥珀)

改造策略

核心原则：把 17 处硬编码深色值替换为已有的 CSS 变量，激活 globals.css 中已定义的浅色令牌体系。

具体动作：
  1. globals.css — 已完备，无需改动（antd 覆盖规则已使用 var()）
  2. AppLayout.tsx — 移除 style={{ background: '#09090F' }}，改用 CSS class
  3. Header.tsx — 移除 background: 'rgba(9,9,15,0.8)'，使用 bg-card class
  4. Sidebar.tsx — 移除渐变硬编码，使用 bg-card + border-subtle
  5. SearchModal.tsx — 移除 #12122A 背景，使用 bg-card
  6. NotificationDropdown.tsx — 同上，移除硬编码深色
  7. 通用：所有 text-white → text-primary，所有 text-gray-* → text-secondary
