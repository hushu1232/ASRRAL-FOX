# AstralFox B端/C端 完成度总评

> 站在使用者日常视角，用大白话评价产品能不能用、好不好用、还差什么  
> 对标：企业级 SaaS 平台（B端）+ 市面桌宠竞品（C端）

---

## 一、B端：创作者 & 管理员平台

### 日常场景走查

#### "我刚注册，前5分钟"

| 步骤 | 体验 | 评分 |
|------|------|------|
| 注册 | 填邮箱+用户名+密码，流畅 | ✅ |
| 登录 | JWT + refreshToken cookie，标准 | ✅ |
| 进仪表盘 | 四张KPI卡片全是0，**没有任何下一步引导** | ❌ |
| 配Pet | 有Setup Wizard，但**要自备Azure+OpenAI密钥** | ⚠️ |
| 逛Market | 空的，没推荐内容 | ❌ |

**问题根因**：空状态零引导。用户注册完不知道要干嘛。

#### "我要上传模型出售"

| 步骤 | 操作 | 问题 |
|------|------|------|
| 1 | 上传素材到资产库 | ✅ 支持分块上传，最大500MB |
| 2 | 创建商品 | ⚠️ 要先上传到资产库，再手动关联路径 |
| 3 | 填表单 | ⚠️ 预览图需手动输入URL，无自动缩略图 |
| 4 | 等审核 | 人工审核，无预计时间提示 |
| 5 | 上线 | 只有5种固定分类 |

**比竞品差距**：Steam创意工坊发布一步到位（拖文件→填描述→定价→发布），多了一步"资产库中转"。

#### "我卖出商品了，钱呢？"

**这是最严重的问题。**

流程是对的：买家点购买→支付宝/微信扣款→Webhook回调→订单状态paid→卖家余额+85元。但是：

```typescript
// market.service.ts:345
pendingPayout: totalRevenue, // simplified — no actual payout system yet
```

- ❌ **没有卖家绑定收款方式页面**（无银行卡/支付宝账号输入）
- ❌ **没有实名认证流程**
- ❌ **没有自动结算周期**
- ❌ **没有卖家卖出通知**（卖出去东西了卖家收不到消息）
- ❌ **没有买家→卖家沟通通道**（即时消息DB表建了但没实现）

卖方视角的真实体验：_"我在平台上有 ¥850 待结算，但我拿不出来，也不知道什么时候能拿，也没人通知我卖掉东西了。"_

#### "我是管理员"

| 能做的 | 不能做的 |
|--------|----------|
| 审核商品 ✅ | 下架已上架商品 ❌ |
| 改用户角色 ✅ | 看用户详情页 ❌ |
| 看审计日志 ✅ | 管理订单/退款 ❌ |
| 配支付渠道 ✅ | 卖家结算管理 ❌ |
| 等级配置 ✅ | 发平台公告 ❌ |

### B端评分

| 场景 | 完成度 | 一句话 |
|------|--------|--------|
| 注册→首用 | 60% | 空状态无引导，BYOK门槛高 |
| 上传→出售 | 70% | 资产库中转多一步，缺自动缩略图 |
| **售出→收款** | **40%** | **提现空壳，拿不到真钱** |
| 自定义宠物 | 80% | 配置完整但受制外部API |
| 管理后台 | 70% | Tab全但缺订单/用户详情 |
| 支付配置 | 85% | 三渠道齐，但入口不明显 |
| **B端综合** | **68%** | |

## 二、C端：桌面宠物用户

### 日常场景走查

#### "早上装好，到晚上关机，狐狸陪了我一整天"

**09:00** 首次启动 → StartupWizard引导 → 跳过API配置 → 选CatTail模型 → 起名"小星" → 完成。狐狸出现在桌面右下角，半透明，微微呼吸起伏。

**09:15** 开始写代码。狐狸检测到VS Code窗口，归类"coding"。空闲时随机做12种动作：挠耳朵、伸懒腰、歪头、蹦跶、挥手…眼睛跟着鼠标转。偶尔说一句"代码写得怎么样了呀~"。在屏幕下半部分慢慢散步，走到边上自动折返。

**10:00** 整点报时："10点了，开始一天的工作吧！" 带着happy表情。

**12:30** 午饭离开。狐狸检测到2分钟无操作→闭眼睡觉。娱乐需求开始掉。

**13:30** 回来。狐狸醒过来："你回来啦！好想你！" 带着surprised表情+跳跳动画。但娱乐需求已经掉到30%以下，过了一会又说"好无聊呀~来聊聊天吧！"

**15:00** 写代码累了，右键点狐狸→换了个碧蓝航线的企业模型。深紫色菜单弹出，10个模型选，企业那个有521个drawables，切的时候卡了半秒。新模型比CatTail精细很多。

**17:00** 点了下语音按钮，说了句"你好呀"。狐狸切换Listening状态，VAD检测到人声，开始录音，发送到后端。3秒后收到LLM回复，TTS播放。说话时嘴型同步开合，眼睛微眯微笑。我说了句"今天好累"，它回"主人辛苦了！摸摸头~"，情绪标签切到happy，PAD愉悦度+0.2。

**23:30** 打Valorant。狐狸检测到前台窗口是游戏+时间是深夜，说"这么晚了还在打游戏！该休息啦，明天再玩嘛~"。配合TimeAwareness的深夜关怀，每隔45分钟又提醒一次。如果你打到凌晨2点，狐狸估计都说了三四遍了。

#### "最爽的5个瞬间"

1. **拖拽+投掷** — 按住狐狸拖走，有弹簧跟随感（不僵硬），松手后狐狸带着惯性飞出去，撞到屏幕底部弹一下再停。像扔一个软软的娃娃。
2. **打断对话** — 狐狸在说话时我插嘴，它立刻停止说、切回听，等我说完再回应。像真人聊天。
3. **歪头杀** — 写代码时狐狸歪头看我，耳朵立起来，眼睛微眯，萌一脸。
4. **右键换装** — 不想用CatTail了，右键→点一下Enterprise就切了，3秒钟。
5. **深夜自动晚安** — 23点准时提醒睡觉，比我自己还清楚时间。

#### "最尬的5个瞬间"

1. **两句话叠着说** — TimeAwareness的整点报时和ProactiveChat的主动搭话撞车："10点了，开始工——你回来啦！好想你！" 听着像个精神分裂。
2. **离开座位的Bug** — 我去了趟厕所10分钟，回来时狐狸说了"我想你"，但没说自己饿了（饿了5分钟了，ProactiveChat下一轮10秒后才反应过来）。
3. **模型列表不对** — 向导里选了某模型，桌面上的还是CatTail（因为拼路径方式不一样）。
4. **网络断了静默挂** — 如果后端没启动，点语音按钮什么反应都没有，没人告诉我"后端没连上"——虽然连接状态灯会红，但作为新用户我不认识那个小点是什么意思。
5. **3个入口3份模型数据** — 向导7个、右键10个、网页11个…对不上。
