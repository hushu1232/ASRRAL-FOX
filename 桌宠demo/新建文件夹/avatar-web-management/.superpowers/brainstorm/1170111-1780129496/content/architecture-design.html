2. 六个核心新增模式

📋 统一页面模式

ListView → 筛选+排序+分页+批量操作
EditView → 表单+校验+乐观提交+撤销
CreateView → 向导分步+实时预览
ShowView → 详情+关联+操作
以约定而非组件形式存在，透传 Ant Design

🎨 主题引擎

基于现有 CSS 变量体系，新增：
• ThemeProvider — 浅色/深色/系统三态
• ConfigProvider 动态 token 注入
• 持久化到 localStorage + cookie (SSR 首屏)
• 平滑过渡动画 (0.3s 背景色过渡)

🔍 筛选器系统

可复用的 <FilterBar> 组件：
• 搜索框 + 多选筛选 + 排序器
• URL query params 同步 (可分享/可收藏)
• 移动端折叠到底部抽屉
• 预设筛选保存

⚡ 乐观更新 UI

react-admin 的 mutate+undo 模式：
• 操作立即反映 UI（SWR optimisticData）
• 失败时 Toast + 撤销按钮
• 批量操作显示进度条
• 离线队列（IndexedDB 暂存）

♿ 可访问性增强

react-admin 级别的 a11y：
• 完整焦点管理（弹窗/抽屉/模态框）
• aria-live 区域标注（动态内容播报）
• 键盘导航标准化（Tab/Arrow/Escape）
• axe-core 集成 CI 检查

🧩 仪表盘小部件

可配置的 Dashboard 系统：
• Widget 注册机制
• 拖拽排序 + 显示/隐藏
• 按角色预设布局
• 数据刷新间隔可配
